# DFUSE ADB GUI 🔧📱

A lightweight ADB control panel built with Python + Tkinter.

## ✨ Features
- List devices
- Reboot (system / recovery / bootloader)
- Install APKs
- ADB shell commands
- ADB Wi-Fi (tcpip + connect)
- Wireless pairing (Android 11+)
- Dark theme UI
- Windows Support Coming Soon
